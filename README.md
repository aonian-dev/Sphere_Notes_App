# SphereNotes

## English (Italian below)

SphereNotes is a visual note-taking application inspired by the Sphere Grid from Final Fantasy X.

Features:

* Create notes as interactive sphere nodes
* Connect notes with visual links
* Drag and organize nodes freely on an infinite canvas
* Customize node symbols and colors
* Persistent storage using SQLite
* Beautiful fantasy-inspired interface with glowing effects

### Requirements

* Python 3.9+
* PyQt5

### Run

```bash
pip install PyQt5
python spherenotes.py
```

All notes and connections are automatically saved in `sphere_notes.db`.

## Italiano

SphereNotes è un'applicazione per prendere appunti, ispirata alla Sphere Grid di Final Fantasy X.

Caratteristiche:

* Creazione di note come sfere interattive
* Collegamenti visivi tra le note
* Organizzazione delle sfere su una tela infinita
* Personalizzazione di simboli e colori
* Salvataggio tramite SQLite
* Interfaccia fantasy con effetti luminosi

### Requisiti

* Python 3.9+
* PyQt5

### Avvio

```bash
pip install PyQt5
python spherenotes.py
```

Tutte le note e le connessioni vengono salvate automaticamente nel file `sphere_notes.db`.
